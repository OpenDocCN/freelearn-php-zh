Appendix C. phpBB Code and Permission Cheatsheets

In this appendix you’ll find both a quick reference for the BBCodes used while posting, and tables that describe the phpBB privileges system. You will find more information about the BBCodes in Chapter 3 and about the permissions in Chapter 5.

BBCode Cheatsheet

The available BB Code tags are divided into four categories based on their purpose—tags used for formatting, for creating lists, for quotes and code snippets, and finally for adding images and links.

Formatting

Code

Results and Comments

This is [b]bold[/b]

...and this is [i]italics [/i].

Now - [u]underline [/u].

[u][i][b]Nesting...[/b][/i][/u]

For nesting tags, close the last tag you opened first, and keep going in the reverse order until the first tag is closed.

[color=darkred]Dark red color using color name[/color]

[color=#000099]Blue, using HTML color code[/color]

[size=1]Hardly visible text[/size]

‘1’ is the smallest possible value for text size, which results in almost invisible text.

[size=12]Normal text[/size]

[size=29]The largest[/size]

‘29’ is the largest font size.

Results and Comments

[list]

[*]Apples

[*]Pears

[*]Bananas

[/list]

This is an example of an unordered list.Every item in the list starts with [*] and ends with a new line.

[list=1]

[*]Apples

[*]Pears

[*]Bananas

[*]Oranges

[/list]

Ordered lists are created just like the unordered ones—by adding an =1 parameter to the list tag. Lists always start from 1, so typing =5 will not make the list start from 5, but will cause the entire list to be displayed as it is, exposing the BB tags.

[list=a]

[*]Apples

[*]Pears

[*]Bananas

[*]Oranges

[/list]

Ordered lists can also use letters instead of numbers.

[*]Oranges

Quotes and Code Snippets

Code

Results and Comments

[quote]Just a quote[/quote]

[quote="The dude"]Quoting another user[/quote]

[quote=" http://mail.yahoo.com "]Free web mail[/quote]

This shows an undocumented behavior in the BBCode parser that allows you to quote a site and at the same time create a link to the site. Notice the spaces surrounding the URL.

[quote=" test@example.com "]I’m a test[/quote]

The same undocumented behavior in phpBB’s BBCode parser allows the creation of an e-mail link.

[code]

if (!$sleeping)

{

keepPosting();

}

[/code]

The code tag is designed to display code snippets, using a fixed-width font.

[code]

--------------

| Sad | Glad |

--------------

| Mad | Bad |

--------------

[/code]

The fixed-width font of the code tag allows posting of indented programming code, but can also be used for posting tabular data and ASCII art.

Inserting Images and Links

Code

Results and Comments

[img]http://www.packtpub.com/images/Packt.png[/img]

The img tag allows you to include images in your posts, provided the image is available on the Internet and you know its URL.

http://www.packtpub.com/

www.packtpub.com

phpBB will convert any string that looks like a URL into a link.

[url]http://www.packtpub.com/[/url]

When using [url] explicitly, the link is not underlined. It uses a CSS style (postlink) that the automatic link conversion does not.

[url=http://www.packtpub.com/]Packt Book Store[/url]

This is the best way to add a link. The text of the link is different from the URL.

[url=http://www.packtpub.com/]

[img]http://www.packtpub.com/images/Packt.png[/img]

[/url]

This is an example of creating a clickable, linked image.

Permissions Cheatsheet

Permissions are discussed a bit in Chapter 3 and then in detail in Chapter 5. The table that follows provides a quick reference to such permissions.

Permission Types

Permission type

Description

View

Users can see that a forum exists on the board’s front page.

Read

Users can list forum topics, read the postings, and view the profiles and contact information of members.

Reply

Users can post their opinions about existing topics.

Post

Users can start new topics.

Edit

Users can edit their own postings.

Delete

Users can delete their own postings, as long as no one has replied to them.

Sticky

Users can start sticky topics.

Announce

Users can post announcements.

Vote

Users can vote in polls.

Poll create

Users can create polls for others to vote in.

Permission Levels

Permission level

Description

ALL

Everybody who accesses the board, including members currently not logged in.

REG

A registered member of the board who has logged in.

PRIVATE

A special permission level that is assigned individually. (Refer to Chapter 5.)

MOD

A moderator who can do everything to a forum.

ADMIN

A board administrator—someone who can access the Administration Panel.

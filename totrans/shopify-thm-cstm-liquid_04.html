Section 2: Exploring Liquid Core

This section will dive into the Liquid core, which consists of three main features – objects, tags, and filters – which will help us set a solid groundwork for our future learning. We will learn how to use the different types of objects at our disposal, combined with Liquid programming logic, to turn static templates into a fully dynamic store. Lastly, through the power of filters, we will learn how to manipulate various types of data and turn even a simple set of data into complex features.

This section comprises the following chapters:

  • Chapter 3, Diving into Liquid Core with Tags
  • Chapter 4, Diving into Liquid Core with Objects
  • Chapter 5, Diving into Liquid Core with Filters

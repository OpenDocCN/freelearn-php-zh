Appendix D. phpBB Database Structure

This appendix describes all the phpBB database tables, listed with their fields, relationships, and brief comments describing each table field. As you know phpBB database tables can have a prefix, specified by you when installing. In this appendix, the default phpbb_ prefix is used. This appendix was created with the help of phpMyAdmin (http://www.phpmyadmin.net).

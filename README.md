## 下载

### Docker

```
docker pull apachecn0/freelearn-php-zh
docker run -tid -p <port>:80 apachecn0/freelearn-php-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g freelearn-php-zh
freelearn-php-zh <port>
# 访问 http://localhost:{port} 查看文档
```
